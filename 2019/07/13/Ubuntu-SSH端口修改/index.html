Ubuntu ssh端口修改

Author: DoffChen Date: July 13, 2019  11:57:24 Category: Ubuntu Linux

SSH修改远程连接端口号

1.第一步:打开SSH配置文件

sudo vim /etc/ssh/sshd_config
(这里使用vim和vi都可以,如果vim没有安装可以自行安装:sudo apt-get install vim)

2.打开配置文件后找到:

Port 22
按i键进入编辑模式
将后面的22端口修改成要设置的端口号
然后按esc->输入:wq!保存
然后重启SSH服务:
sudo service ssh restart

3.最后可以查看SSH服务监听的端口:

sudo netstat -tunlp | grep ssh
